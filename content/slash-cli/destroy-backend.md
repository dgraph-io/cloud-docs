Destroy a Backend by id.

USAGE
```
  $ slash-graphql destroy-backend ID
```

ARGUMENTS
```
  ID  Backend id
```

OPTIONS
```
  -q, --quiet    Quiet Output
  -y, --confirm  Skip Confirmation
```

EXAMPLE
```
  $ slash-graphql destroy-backend "0xid"
```
