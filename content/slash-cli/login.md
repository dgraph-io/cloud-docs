Login to Slash GraphQL. Calling this function will keep you logged in for 24 hours, and you will not need to pass access tokens for any backends that you own.

USAGE
```
  $ slash-graphql login
```

OPTIONS
```
  -q, --quiet  Quiet Output
```

EXAMPLE
```
  $ slash-graphql login
```
