Logout of Slash GraphQL Command Line.

USAGE
'''
  $ slash-graphql logout
'''

OPTIONS
'''
  -a, --all    Log out of all command line clients
  -q, --quiet  Quiet Output
'''

EXAMPLES
'''
  $ slash-graphql logout
  $ slash-graphql logout -a
'''
