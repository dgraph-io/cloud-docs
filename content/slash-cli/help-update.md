### Help

Display help for the `slash-graphql` CLI.

USAGE
```
  $ slash-graphql help [COMMAND]
```

ARGUMENTS
```
  COMMAND  Command to show help for
```

OPTIONS
```
  --all  See all commands in CLI
```

### Update CLI

Update the `slash-graphql` CLI.

USAGE
```
  $ slash-graphql update [CHANNEL]
```
